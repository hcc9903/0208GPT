# 在手机上使用 APP 的实施计划

本项目包含两个主要应用：一个是基于 Python Streamlit 的 **DEBIT_SYNC 智能对账工具**（Web 应用），另一个是 **记帐鸭**（Android 原生应用）。为了能在手机上使用它们，我们需要分别采取不同的策略。

## 1. 使用 DEBIT_SYNC 对账工具 (Web 应用)

这种方式最简单，只需通过局域网连接即可在手机浏览器上访问。

### 实施步骤：
1. **启动服务**：在电脑终端运行程序。
2. **获取内网地址**：确认电脑在局域网中的 IP 地址（已确认：`192.168.1.9`）。
3. **手机连接**：手机连接与电脑相同的 WiFi。
4. **访问**：在手机浏览器输入 `http://192.168.1.9:8501`。

---

## 2. 使用记帐鸭 (Android 应用)

这是一个原生 Android 项目，需要编译为 APK 文件后安装到手机。

### 实施步骤：
1. **环境检查**：检查本地是否安装了 Java (JDK) 和 Android SDK。
2. **编译 APK**：使用 `gradlew` 构建工具生成通用的调试版或正式版 APK。
3. **传输安装**：将生成的 `.apk` 文件通过 USB 或第三方传输工具发到手机。
4. **安装**：在手机上点击 APK 并允许安装来自未知来源的应用。
